FHIR RESTful Capabilities

Mode: client

El Cliente Analiste del Repositorio de Biopsia del Ministerio de Salud permite extraer los datos de los informes de anatomía patológica y sus resultados para realizar análisis secundario y tomar decisiones de salud pública.

Security
Enable CORS: no
Security services supported: OAuth

OAuth2 usando el APIG del Ministerio de Salud para generar Tokens para enviar y acceder al respositorio

Summary of System-wide Operations
ConformanceOperationDocumentation
SHALL$export

Operación para exportar los datos del repositorio de informes de anatomia en formato FHIR Bulk Data, para analisis secundario
